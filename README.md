# 🍽️ DishDash Recipes

A modern and interactive recipe website built using **HTML, CSS, and JavaScript**.
Users can explore different dishes, search recipes, and view detailed ingredients and procedures.

---

## 🚀 Features

* 🔍 Search functionality for recipes
* 🍕 Category-based filtering (Pizza, Burger, Pasta, etc.)
* 🎨 Clean and responsive UI
* 📺 YouTube tutorial links for recipes
* 👨‍💻 Team contribution section with social links

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
